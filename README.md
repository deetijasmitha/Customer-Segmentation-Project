# Customer Segmentation using K-Means Clustering

## Project Overview

This project focuses on segmenting customers based on their purchasing behavior and demographic information using Machine Learning techniques. The main objective is to identify different customer groups that can help businesses make data-driven marketing decisions and improve customer targeting strategies.


## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook (VS Code)



## Project Workflow

1. Data Collection and Loading
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Applying K-Means Clustering
6. Determining Optimal Clusters using Elbow Method
7. Data Visualization
8. Customer Insights and Recommendations



## Key Features

* Performed customer segmentation using K-Means clustering
* Analyzed customer purchase behavior and spending patterns
* Visualized customer groups using scatter plots
* Identified high-value and low-value customer segments
* Generated business insights for targeted marketing



## Insights Generated

* High-income customers with high spending scores are premium customers and ideal targets for luxury products and exclusive offers.
* Customers with low income and low spending behavior require discount-based marketing strategies.
* Some customers have high income but low spending scores, indicating opportunities for personalized engagement campaigns.
* Average-income customers form the largest customer segment and contribute to stable business revenue.
* Customer segmentation helps businesses improve marketing efficiency and customer retention.



## Business Recommendations

* Create personalized offers for premium customers.
* Introduce loyalty programs for frequent buyers.
* Use targeted advertisements for different customer groups.
* Focus retention strategies on high-value customers.



## Project Outcome

Successfully developed a machine learning-based customer segmentation system that provides meaningful customer insights and supports better business decision-making through data analytics.
